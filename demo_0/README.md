# 引入glfw和glad

    glfw3和glad直接用编译好的
    
1.新建空c++项目 文件 -> 新建项目
    注意修改项目名和路劲

2.在解决方案属性里加入库
    项目属性面板打开，VC++ -> 包含目录和库目录中分别添加头文件目录和库文件目录
    连接器 -> 输入 添加glfw3.lib 和 opengl.lib
    项目中添加外部文件 项目 -> 添加现有项 -> glad.c

3.新建入口文件
    项目 -> 添加类 main
    引用库
    ```
        #include <glad/glad.h>
        #include <GLFW/glfw3.h>
    ```

4.代码部分直接复制，点击运行，正确显示一个黑乎乎的窗口
